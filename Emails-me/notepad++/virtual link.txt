Installing fitConnectModule

1. clone the repository https://github.com/stefanka-lingerie/FitConnect-ng4.git
2. in both folder install npm
3. in folder FitConnectModule, do -> npm run build
4. in folder FitConnectDevelopmentApp do -> npm link fitconnectmodule
